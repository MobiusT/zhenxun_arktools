### 前言

原插件是 -> **[这个](https://github.com/NumberSir/nonebot_plugin_arktools)** <- 

不使用真寻的请前往原插件仓库

本插件无法单独使用，必须装到 **[真寻bot](https://github.com/HibiKier/zhenxun_bot)** 中才能正常运行

### 安装方法

将 **arktools** 文件夹整个放进真寻 bot 的 **plugins** 文件夹即可
